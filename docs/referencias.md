# Referencias y portales

## Portales de descarga de datos (gratis)

| Portal | Qué tiene | Acceso |
|---|---|---|
| **Copernicus Data Space Ecosystem** | Sentinel-1/2/3/5P, navegador y API | [dataspace.copernicus.eu](https://dataspace.copernicus.eu) |
| **ASF DAAC (Alaska)** | Sentinel-1, ALOS, y **HyP3** (InSAR en la nube) | [search.asf.alaska.edu](https://search.asf.alaska.edu) |
| **USGS EarthExplorer** | Landsat, ASTER, SRTM, declassified | [earthexplorer.usgs.gov](https://earthexplorer.usgs.gov) |
| **NASA Earthdata Search** | MODIS, VIIRS, SMAP, GRACE, EMIT, NASADEM | [search.earthdata.nasa.gov](https://search.earthdata.nasa.gov) |
| **Copernicus DEM** | DEM global GLO-30 (de TanDEM-X) | [AWS Open Data](https://registry.opendata.aws/copernicus-dem/) |
| **OpenTopography** | SRTM, Copernicus DEM, LiDAR | [opentopography.org](https://opentopography.org) |
| **CONAE** | SAOCOM (banda L), registro | [conae.gov.ar](https://www.conae.gov.ar) |
| **ASI / PRISMA** | Hiperespectral, registro | [prisma.asi.it](https://prisma.asi.it) |
| **NOAA GOES (AWS)** | GOES geoestacionario, casi en tiempo real | [registry.opendata.aws/noaa-goes](https://registry.opendata.aws/noaa-goes/) |

## Datos orbitales (TLE) para el globo

- **Celestrak** — elementos orbitales (TLE/GP) actualizados: [celestrak.org](https://celestrak.org)
- **Space-Track** (NORAD, requiere registro): [space-track.org](https://www.space-track.org)

## Herramientas de software (libres)

- **HyP3** (ASF) — interferogramas Sentinel-1 en la nube: [hyp3-docs.asf.alaska.edu](https://hyp3-docs.asf.alaska.edu)
- **MintPy** — series de tiempo InSAR (SBAS): [github.com/insarlab/MintPy](https://github.com/insarlab/MintPy)
- **Ames Stereo Pipeline (ASP)** — DEM por estéreo: [stereopipeline.readthedocs.io](https://stereopipeline.readthedocs.io)
- **QGIS** / **GDAL** — SIG y manejo de raster: [qgis.org](https://qgis.org) · [gdal.org](https://gdal.org)
- **satellite.js** — propagación SGP4 en el navegador: [github.com/shashwatak/satellite-js](https://github.com/shashwatak/satellite-js)
- **Skyfield** — efemérides y órbitas en Python: [rhodesmill.org/skyfield](https://rhodesmill.org/skyfield/)

## Para aprender más

- **ESA EO College** — cursos gratuitos de teledetección: [eo-college.org](https://eo-college.org)
- **NASA ARSET** — capacitación aplicada: [appliedsciences.nasa.gov/arset](https://appliedsciences.nasa.gov/get-involved/training/arset)
- **Copernicus** — documentación de misiones: [copernicus.eu](https://www.copernicus.eu)
